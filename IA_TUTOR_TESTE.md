## 🧠 Sistema Inteligente de Contexto

A nova IA usa **detecção semântica avançada**:

```
EXEMPLO 1:
Entrada: "Como funciona a meiose?"
Detectado: BIOLOGIA
Resposta: Explicação detalhada sobre meiose com contexto biológico

EXEMPLO 2:
Entrada: "Qual é a fórmula de velocidade?"
Detectado: FÍSICA
Resposta: Com aplicações práticas em mecânica

EXEMPLO 3:
Entrada: "Me explique sobre a Revolução Francesa"
Detectado: HISTÓRIA
Resposta: Contexto histórico e impactos
```

## 🔧 Funções JavaScript Principais

1. `aiMode(modo)` - Altera modo de operação
2. `sendAIMessage()` - Envia mensagem para IA
3. `generateAIResponse(msg)` - Gera resposta
4. `addAIMessage(txt, role)` - Adiciona msg ao chat
5. `updateAIStats()` - Atualiza painel de stats
6. `loadFromHistory(idx)` - Carrega histórico
7. `clearAIChat()` - Limpa conversação
